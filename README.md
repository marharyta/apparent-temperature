Formulas used:

for Kelvin
T(K) = T(°C) + 273.15
T(°C) = T(K) - 273.15
T(K) = (T(°F) + 459.67)× 5/9
T(°F) = T(K) × 9/5 - 459.67
T(°F) = T(°C) × 9/5 + 32
T(°F) = T(°C) × 1.8 + 32
T(°C) = (T(°F) - 32) × 5/9
T(°C) = (T(°F) - 32) / (9/5)
T(°C) = (T(°F) - 32) / 1.8

T(°F) = T(°C) × 9/5 + 32

or

T(°F) = T(°C) × 1.8 + 32
